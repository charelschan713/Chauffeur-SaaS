# Platform Backend

Multi-tenant SaaS chauffeur booking platform backend (Blacklane-like), built with NestJS + TypeScript.

## Planned stack
- NestJS + TypeScript
- PostgreSQL + TypeORM
- Redis
- JWT + Refresh Token
- Stripe
- AWS S3
- SendGrid
- Docker / docker-compose

## Branch strategy
- `main`: production
- `staging`: pre-production / QA
- `dev`: active development

## Getting started
```bash
cp .env.example .env
npm install
npm run start:dev
```
